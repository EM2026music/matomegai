========================================
 まとめ買いリスト
 1週間の献立から買い物リストを自動で作るアプリ
========================================

【使い方（かんたん）】

 1) この中の「index.html」をダブルクリックする
    → ブラウザが開いて、すぐ使えます

 2) 使い方がわからないときは
    「guide.html」をダブルクリック
    → 画面写真つきの説明書が開きます


【スマホで使いたいとき】

 ZIPを解凍しなくても、下のリンクから直接使えます。
 こちらのほうが便利です（オフラインでも動きます）。

   アプリ  : https://em2026music.github.io/matomegai/
   使い方  : https://em2026music.github.io/matomegai/guide.html

 スマホでは、ブラウザで上のアプリのリンクを開いてから
   ・iPhone : 共有ボタン → 「ホーム画面に追加」
   ・Android: 右上の「⋮」 → 「ホーム画面に追加」
 とすると、普通のアプリのように使えます。


【大事なこと】

 ・お金はかかりません。広告もありません。
 ・登録もログインも不要です。
 ・入力したデータ（献立・買い物リストなど）は
   お使いの端末の中だけに保存されます。
   インターネット上には一切送られません。
 ・データを他の端末に移したいときは、アプリ内の
   「⚙設定」→「バックアップ保存」でファイルを保存し、
   移動先で「読み込み」してください。


【同梱ファイル】

 index.html ......... アプリ本体（これを開く）
 guide.html ......... 使い方ガイド
 guide-*.png ........ ガイド用の画面写真
 manifest.json ...... アプリとして登録するための設定
 sw.js .............. オフラインで動かすための仕組み
 icon-*.png ......... アプリのアイコン
 README.txt ......... このファイル

 ※ manifest.json / sw.js / icon-*.png は
   スマホで「アプリとして」使うためのファイルです。
   パソコンでindex.htmlを開くだけなら使いません。
